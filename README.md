## Deploy on GitHub Pages
1. Create a repo (or use an existing one — this can live alongside `index.html` from the first version, e.g. rename to `case-file.html`, or put it in its own repo).
2. Upload `index-v2.html` to the repo root (rename to `index.html` if it should be the homepage).
3. Settings → Pages → Deploy from branch → `main` → `/ (root)`.
4. Live at `https://<your-username>.github.io/<repo-name>/`.
